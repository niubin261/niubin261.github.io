mapreduce

mapreduce是一种架构层次上的解决海量数据处理的编程模型

概念

mapreduce分为两个过程，分别为map和reduce

map

将大数据集进行分解操作得到两个或者更小的桶。每台处理器对分割出来的每个桶进行操作，获取一组中间值
domap读取文件根据文件的内容得到一组key,value值，并且根据这组key,value值的hash将其分为不同的文件中hash的时候根据reduce的个数将文件分到不同的reduce中//隐藏
map是将数据{key:value}写成{key:value}的形式

reduce

reduce就是把这些中间结果通过一定的函数进行处理得到最终的答案
doreduce是将{key:value}转换为{key:list{value}}//将文件中的key：value根据key对value进行整合//隐藏
reduce将{key:list{value}}写成{key:value}//将list整合为结果

mapreduce

在编写mapreduce程序的时候一般domap以及doreduce不需要直接编写，只需要编写map以及reduce函数即可，以供domap和doreduce调用

golang中的标签

select
Go语言会从头至尾的判断每一个case中的发送和接收语句。如果其中任何一条语句可以执行（即没有被阻塞），那就从那些可执行的语句中任意选择一条来使用。如果所有的通道都被阻塞，那可能有两种情况。第一种，如果有default语句，那就会执行default 语句，同时程序的执行会从select语句恢复。第二种，如果没有default语句，则select语句会一直阻塞，直到有一个通道可用

后缀表达式

第一个栈维护操作数
第二个栈维护操作符
当操作数是进入第一个
当操作符是进入第二个，如果此时栈顶也是操作符，栈顶出栈使其进入栈1，之后该操作数入栈2
如果是括号，如果是左括号则进栈
如果是右括号：
则栈顶出栈直到遇到左括号，出栈数进入栈1，并且左括号也要出栈

go语言中的rpc
Go RPC的函数只有符合下面的条件才能被远程访问，不然会被忽略，详细的要求如下：

  1. 函数必须是导出的(首字母大写)
  2. 必须有两个导出类型的参数，
  3. 第一个参数是接收的参数，第二个参数是返回给客户端的参数，第二个参数必须是指针类型的
  4. 函数还要有一个返回值error

golang中的指针
new(T) 返回 T 的指针 T 并指向 T 的零值。
make(T) 返回的初始化的 T，只能用于 slice，map，channel。
指针的一大用途就是可以将变量的指针作为实参传递给函数，从而在函数内部能够直接修改实参所指向的变量值

使不使用结构体指针和使不使用指针的出发点是一样的，那就是你是否试图在函数内部改变传递进来的参数的值
